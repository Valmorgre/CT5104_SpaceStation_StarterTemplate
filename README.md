# UoG CT5104 – VR / AR Space Station Template

This repository is the **starter Unity project** for the CT5104 VR/AR module.
Each student will receive **their own private copy** via zip download.

---

## Unity Version
**Unity 6 LTS (6000.0.53f1)**  
Please use the same version to avoid compatibility issues.

---

## Required Packages
These are already configured in the project:
- XR Interaction Toolkit
- OpenXR
- Input System

Unity will prompt to install missing packages automatically.

1. Go to Code > Download zip
2. DO NOT "fork" from this repo or create a new branch
3. Extract to a new project folder
4. Open **Unity Hub**
5. Click **Open → Add project from disk**
6. Select this folder
7. Allow Unity to regenerate Library files
8. Open the main scene when prompted
9. Use Github Desktop to create a new repo for yourself

Do **not** commit the `Library/` folder.
